# Ora Client for Java

This is a simple synchronous client for accessing the ORA rest API.

## To Play

* Clone
* mvn compile
* mvn test

Take a look at the tests to see usage.

## To Include in your maven project:

    <dependency>
      <groupId>systems.ora</groupId>
      <artifactId>ora-client</artifactId>
      <version>0.7/version>
    </dependency>
